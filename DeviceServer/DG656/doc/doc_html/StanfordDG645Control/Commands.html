StanfordDG645Control Class Commands
Name
Input type
Output type
Level
Description
State DEV_VOID DEV_STATE OPERATOR This command gets the device state (stored in its device_state data member) and returns it to the caller.
Status DEV_VOID CONST_DEV_STRING OPERATOR This command gets the device status (stored in its device_status data member) and returns it to the caller.
SingleShot DEV_VOID DEV_VOID OPERATOR When the DG645 is configured for single shot triggers, this command initiates a single trigger. When it is configured for externally triggered single shots, this command arms the DG645 to trigger on the next detected external trigger.
LoadDefaults DEV_VOID DEV_VOID OPERATOR Resets the device to it`s default UI_settings.
LoadSettings DEV_USHORT DEV_VOID OPERATOR Reads back the previously saved instrument UI_settings from location [1..9].
SaveSettings DEV_USHORT DEV_VOID OPERATOR Saves the current instrument UI_settings to location [1..9].
WriteRead DEV_STRING DEV_STRING OPERATOR Writes the argument string to the delay generator and returns the answer.
